## 프로젝트 소개
대학교 별로 나뉘어져 있는 커뮤니티를 한곳으로 모아 전국 대학생들이 이야기를 나눌수 있는 커뮤니티를 만들기로 한다.
<br><br>
- 시스템 구성도

## 주요 기능
- 온라인 동아리 및 모임
  - 동아리나 모임을 만들어 마음이 맞는 사람들끼리 소통할 수 있음
- 공모전 모임
  -  나가고 싶은 공모전에 대한 팀을 구하기 위해 모임을 만들고 팀원을 구할 수 있음
- 모임 내 채팅 및 공부 시간 확인
  - 같은 모임 사람끼리 채팅 및 공부한 시간을 확인 과 시간으로 인한 순위도 얻을 수 있음
